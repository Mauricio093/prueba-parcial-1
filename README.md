Sistema de Gestión de Usuarios

Aplicación de consola en *Java* para registrar y listar usuarios.  
La información se guarda en un archivo 📄 usuarios.txt.

---

🚀 Funcionalidades

✅ Registrar usuario (nombre y correo)  
📋 Listar usuarios registrados  
💾 Guardado en archivo de texto  

▶️ Instrucciones de ejecución

1️⃣ Abrir el proyecto en NetBeans.
2️⃣ Ejecutar la clase app.java.
3️⃣ Leer el menú y seleccionar una de las tres opciones.
4️⃣ Ingresar el nombre de la persona.
5️⃣ Ingresar el correo electrónico.

 

📂 El archivo usuarios.txt se crea automáticamente en la carpeta del proyecto.
